# Ozon Parser

Парсер поисковой выдачи Ozon для мониторинга позиций товаров.

## 📋 Требования

- Python 3.10+
- Playwright
- Git

## 🚀 Установка

```bash
# Клонируем репозиторий
git clone https://github.com/your-username/ozon-parser-test.git
cd ozon-parser-test

# Устанавливаем зависимости
pip install -r requirements.txt

# Устанавливаем браузеры для Playwright
playwright install chromium
